Vim

Vim has been my de facto text editor since I started using linux. Reading about all the hype around Vim and Emacs, I was initially confused between what to choose. I thought of trying both the editors and then deciding which one to use. But it so happened that I started using Vim, learnt its commands, and forgot about Emacs. That has been my story.

I make some very limited use of the power of Vim. The few keys which I use extensively are: 

j   // for down
k   // for right
h   // for left
l   // for right
x   // to delete the character at the cursor
r   // to replace the character at the cursor
dd  // to cut the line
yy  // to copy the line
[num]D  // to cut num lines from the current cursor position
[num]Y  // to copy num lines from the current cursor position
i   // to start typing at current position
a   // to start typing at after the current position
A   // to start typing at the end of the line
u   // undo
Ctrl-r  // redo

I also use some sed commands: 

:%s/text_to_search/text_to_replace/  // this replaces all the occurences of text_to_search
:s/text_to_search/text_to_replace/   // this replaces all the occurences of text_to_search in the current line
/text_to_find   // searches the text in the document
n   // to navigate forward in the search results
N   // to navigate backwards in the search results
:[num]  // go to the line number num
*   // to search the current word on which the cursor is present

I have also made minimal modification to the .vimrc 

set autoindent
set nu
set expandtab
set shiftwidth=4
set softtabstop=4
