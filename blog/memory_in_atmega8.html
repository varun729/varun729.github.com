Memory in Atmega8

Features:

  • 8 Kilobytes of In-System self-programmable flash program memory
  • 512 bytes EEPROM
  • 1 Kilobyte internal SRAM
  • Write/erase cycles : 10000 Flash/100000 EEPROM
  • Optional boot code section with independent lock bits
The Atmega8 instructions are either 16 bit or 32 bit. The program memory is thus organized as 4K * 16 bits. Thus, the program counter(PC) is 12 bits wide for the 4K program memory locations.
