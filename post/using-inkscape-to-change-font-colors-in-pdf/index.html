Using Inkscape to Change Font Colors in PDF

Inkscape is a pretty handy tool to edit vector graphics. PDF files are vector images, and thus can be edited using Inkscape.

I recently found an image that I generated from Excel, where I forgot to change the font color to black. Now that I have the image in the perfect size, I don’t want to go back to Excel and fiddle with the settings all over again. Following are the steps that I take to change the font color.

Step 1. Open the PDF image in Inkscape with the \“very fine\” setting in the \“PDF Import Settings.

Step 2. Select the component whose color you want to change, and then select desired color from the Pallette.

At the bottom of the page is \“Pallette\” (View > Show/Hide > Pallette, in case you don’t see the pallette).
