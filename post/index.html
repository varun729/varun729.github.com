Betweeness Algorithm

Graph theory is interesting. My current project involves improving the performance of graph algorithms by converting memory accesses to random addresses into memory accesses to sequential memory. This improves the memory access latency, and thus improves performance of the application. Because graph algorithms access nearby nodes, sometimes it is not possible to rearrange the memory to make the accesses sequential. For such cases, we can prefetch the memory contents from memory as we perform computation.
