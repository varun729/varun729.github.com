JOS Lab

The most significant part of the course is the lab work. The whole duration of the course is divided up into 7 labs. Each of which involves implementing one part of an operating system. We use JOS as the base.

Following are the components of the labs which have been done so far:

Setting up bootloader

We are supposed to write parts of code for the bootloader. This lab helps in understanding

  • How does a system start up?
  • What is the first instruction which is executed?
  • Difference between the real mode and long mode.
  • How the system start executing kernel code from the bootloader?
