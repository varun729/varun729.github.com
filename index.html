Key Academic Projects

1.
Indexing of the Reference Genome [October 2011 - Present]
Guided by Dr. Ramesh Hariharan, Founder, Strand Life Sciences
  • Burrows-Wheeler Transform of the reference is used as the index. The indexing was done after removing all the N’s(unknown characters) from the reference. The N’s are stored separately, and are used only while reconstructing the reference from the BWT.
  • The BWT index is used for aligning reads(sub-sequence of the genome) on the reference.
  • Further work involves alignment of spliced reads, parts of which may align to different regions in the reference genome.
2. Designing ontology for Interaction Database [August 2011 - Present]
Guided by Dr. Vamsi Veeramachaneni, Vice President, Strand Life Sciences
  • Designed an ontology for biological interactions. The data is stored in a Resource Description Framework(RDF) model defined by the ontology.
  • In comparison to querying the relational database, it is now very easy to make logical queries to the data to see if their are any anomalies. An example is to look for two entities which are up-regulating each other.
3.B.Tech Project : Simulation of radars [September 2009 - April 2010]
Guided by Prof. A.R. Harish, Associate Professor, EE, IIT Kanpur
  • Modeled a glacier surface, and simulated microwaves to estimate the height of the glacier.
  • The irregular surface model of the glacier simulated the surface scattering from an actual glacier very closely.
  • Finite Difference Time Domain(FDTD) technique was used to solve the Maxwell’s equations.
  • This simulation tool can be useful in studying the results of the ongoing research in finding glacier heights in Greenland and other icy regions.
4.Summer Project : Organic LED Simulation [May 2008 - July 2008]
Guided by Prof. Deepak Gupta, Professor, Material Science Engineering, IIT Kanpur
  • Solved partial differential equations numerically, to find the current flowing in the LED under steady state. Using the simulation, current-voltage(I-V) graphs of several organic materials was studied.
